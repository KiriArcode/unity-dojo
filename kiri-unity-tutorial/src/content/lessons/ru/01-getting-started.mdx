В этом уроке мы установим Unity Hub — точку входа для всех проектов Unity.

## Зачем нужен Unity Hub

Unity Hub позволяет:
- Устанавливать и переключать версии редактора Unity
- Создавать и открывать проекты
- Быстро получать обновления

## Шаги установки

1. Перейди на [unity.com/download](https://unity.com/download) и скачай Unity Hub.
2. Установи Unity Hub (macOS / Windows).
3. Запусти Hub и войди или зарегистрируйся в Unity ID.
4. В разделе **Installs** нажми **Install Editor** и выбери версию LTS (рекомендуется для 2D).

После установки редактора можно переходить к созданию первого проекта — об этом следующий урок.
